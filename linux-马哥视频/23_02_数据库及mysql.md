####RDBMS

`R`elational `D`ataBase `M`anagement `S`ystem, 关系型数据库管理系统.

**具备的功能**

* 数据库创建,删除,修改
* 创建表,删除表,修改表
* 索引的创建,删除
* 用户和权限的管理
* 数据的增,删,改
* 查询

**DML**

数据操作语言

`D`ata	`M`anapulate `L`anguage 

* INSERT
* REPLACE
* UPDATE
* DELETE

**DDL**

数据定义语言

`D`ata `D`efination `L`anguage 

* CREATE (创建数据库,表,索引..)
* ALTER (修改数据库,表,索引..)
* DROP (删除数据库,表,索引..)

**DCL**

数据控制语言

`D`ata `C`ontrol `L`anguage

* GRANT (授权)
* REVOKE (收回授权)
