# 13 查找

### 结识查找命令

**`/`**用于正向查找匹配.

**`?`**用于反向查找匹配.

`Vim`会从当前光标位置开始向下扫描, 停在所找到的第一个处匹配上. 查找命令抵达文档结尾处时会回绕至文档开头继续查找.

如果不想绕回文档继续查找, 可以**关闭`wrapscan`**选项.

**`n`**跳转到**下一处**匹配.

**`N`**跳转到**上一处**匹配.

当查找提示符**`/`或`?`**出现时, 按箭头`UP`可以回溯查找历史.

### 高亮查找匹配

**`hlsearch`**, 所有匹配将被高亮起来.

```
:set hls 启用
```

永久禁用.

```
:set nohlsearch 禁用
:set hls! 禁用
:set nohls 禁用
```

临时一次禁用, 直到执行新的或重复的查找命令为止.

```
:nohlsearch
:noh
```
