#VIM 常用操作命令

---

**指定多行删除**

VI中的多行删除与复制
法一：
单行删除，：1（待删除行）d
多行删除 ，：1,10d

**批量添加注释**

在10 - 20行添加 // 注释

		:10,20s#^#//#g
		
在10 - 20行删除 // 注释

		:10,20s#^//##g
		
在10 - 20行添加 # 注释

		:10,20s/^/#/g
		
在10 - 20行删除 # 注释

		:10,20s/^/#/g
		

**多行复制**

复制`9-15`行, 黏贴到`16`行.

		:9，15 copy 16  或 :9，15 co 16
		
**多行剪切**

剪切`9-15`行, 黏贴到`16`行.

		:9，15 move 16  或 :9，15 m 16
		
**批量查找**

		:vimgrep /搜索内容/*
		
* `:cnext(:cn)`         下一个匹配位置
* `:cprevious(:cp)`     上一个匹配位置
* `:cwindow(:cw)`     	 quickfix窗口，可以选择匹配的文件位置
* `:cl(:clist)`         查看所有匹配的位置

**快捷操作文件**

		:new 文件名.后缀 新建文件
		
		:e 文件名 打开文件
		
		:w 文件名 保存文件
		
		:wq	保存呢并推出
		
		:x 退出, 如果文件更改则保存
